# 🚀 TaskFlow Pro - Team Task Manager

A full-stack MERN application for team task management with role-based access control, real-time dashboards, and project tracking.

## ✨ Features

- 🔐 **JWT Authentication** - Secure login/register system
- 👥 **Role-Based Access** - Admin and Member roles
- 📊 **Interactive Dashboard** - Charts, statistics, and analytics
- 📁 **Project Management** - Create and manage multiple projects
- ✅ **Task Tracking** - Kanban-style task management
- 👤 **Team Management** - View and manage team members
- 🎨 **Modern UI** - Material-UI with Framer Motion animations
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 📱 **Responsive Design** - Works on all devices

## 🛠️ Tech Stack

- **Frontend:** React 18, Material-UI 5, Framer Motion, Recharts
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT + bcryptjs
- **Deployment:** Render
